# Daily Focus Planner

A minimalist yet refined daily productivity planner built with Vanilla JavaScript and CSS variable architecture.

This project focuses on clean state management, thoughtful UI interactions, and theme persistence without using any frameworks.

---

## Features

- Add, complete, and delete tasks
- Persistent task storage using localStorage
- Daily automatic reset logic
- Main focus and reflection saving
- Light / Dark mode toggle with persistent theme state
- Animated section focus indicators
- Subtle task micro-interactions
- Responsive layout

---

## Technical Highlights

- Vanilla JavaScript (no libraries)
- CSS variable-based theme architecture
- LocalStorage state persistence
- Dynamic DOM manipulation
- UI state synchronization
- Clean component-style section structure

---

## Design Approach

The goal of this project was to create a focused, distraction-free planner while maintaining a premium feel through:

- Soft micro-animations
- Structured spacing
- Controlled accent glow
- Theme consistency

Rather than building a feature-heavy application, this planner prioritizes intentional UX decisions and clean architecture.
